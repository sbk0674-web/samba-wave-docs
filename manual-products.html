상품관리 매뉴얼

수집된 상품을 편집하고 마켓에 전송하는 전체 워크플로우

1. 상품 목록 보기

상품관리 페이지에서는 3가지 보기 모드를 지원합니다.

간단보기

상품의 핵심 정보(이미지, 상품명, 원가, 판매가, 상태)만 간결하게 표시합니다. 대량 상품을 빠르게 스캔할 때 유용합니다.

건별보기

상품 하나하나의 상세 정보를 모두 표시합니다. 옵션, 설명, 이미지 등 모든 필드를 확인할 수 있습니다.

이미지만보기

상품 이미지를 그리드 형태로 표시합니다. 시각적으로 상품을 확인하고 선택할 때 편리합니다.

[스크린샷] 보기 모드 전환 버튼 — 간단보기 / 건별보기 / 이미지만보기 탭 전환 영역
TIP 대량 상품 관리 시 간단보기 모드를 사용하면 로딩 속도가 빠릅니다.

2. 상품 검색

다양한 조건으로 상품을 검색할 수 있습니다.

검색 유형

  • 상품명 — 원 상품명 또는 등록 상품명에서 키워드를 검색합니다.
  • 상품명+등록명 — 원 상품명과 마켓 등록명을 동시에 검색합니다.
  • 그룹 — 특정 수집 그룹에 속한 상품만 표시합니다.
  • 상품번호 — 소싱처의 상품 고유번호로 검색합니다.
  • 정책 — 특정 정책이 적용된 상품만 표시합니다.
[스크린샷] 검색 필터 영역 — 드롭다운으로 검색 유형을 선택하고 키워드를 입력하는 필드
  1. 검색 유형 드롭다운에서 원하는 검색 조건을 선택합니다.
  2. 검색어를 입력합니다.
  3. 엔터 또는 검색 버튼을 클릭하면 필터링된 결과가 표시됩니다.

3. 상품 편집

수집된 상품의 정보를 수정하여 마켓에 등록할 이름과 설명을 변경합니다.

편집 가능 항목

  • 원 상품명 — 소싱처에서 가져온 원래 상품명 (수정 시 원본과 다르게 관리)
  • 등록 상품명 — 마켓에 등록될 상품명. 마켓별 규정에 맞게 수정합니다.
  • 영문 상품명 — 해외 마켓(아마존, 라쿠텐 등) 전송 시 사용됩니다.
  • 일문 상품명 — 라쿠텐 등 일본 마켓 전송 시 사용됩니다.
[스크린샷] 상품 편집 화면 — 상품명, 등록명, 영문명, 일문명 입력 필드가 표시된 편집 폼
참고 SSG(신세계몰)은 계약 브랜드만 등록 가능하며, 상품명에 브랜드명을 포함할 수 없는 규칙이 있습니다.

4. 정책 적용

상품에 가격 정책을 적용하여 판매가를 자동으로 계산합니다.

  1. 상품을 선택합니다. (개별 선택 또는 전체 선택)
  2. 상단 메뉴의 정책 적용 버튼을 클릭합니다.
  3. 적용할 정책을 드롭다운에서 선택합니다.
  4. 선택한 상품에 정책이 적용되어 마켓별 판매가가 자동 계산됩니다.

마켓별 판매가 확인

정책이 적용된 상품의 건별보기에서 각 마켓별 판매가를 확인할 수 있습니다.

[스크린샷] 마켓별 판매가 표시 — 스마트스토어, 쿠팡, 11번가 등 각 마켓의 계산된 판매가 목록
TIP 정책을 수정하면 해당 정책이 적용된 모든 상품의 판매가가 자동으로 재계산됩니다.

5. AI 기능

AI를 활용하여 상품 정보를 자동으로 가공합니다.

AI 상품명 변경

AI가 마켓 규정에 맞게 상품명을 자동으로 재구성합니다. 브랜드명 위치 조정, 불필요한 문구 제거 등을 수행합니다.

AI 태그 생성

상품 특성을 분석하여 검색에 유리한 태그를 자동 생성합니다.

이미지 변환

상품 이미지의 배경 제거, 워터마크 추가, 리사이즈 등 이미지 가공 기능입니다.

영상 생성

상품 이미지들을 조합하여 간단한 상품 소개 영상을 자동 생성합니다.

[스크린샷] AI 기능 메뉴 — AI 상품명변경, AI태그, 이미지변환, 영상생성 버튼이 나열된 도구 모음
  1. AI 기능을 적용할 상품을 선택합니다.
  2. 상단 도구 모음에서 원하는 AI 기능 버튼을 클릭합니다.
  3. AI가 처리를 완료하면 결과를 확인하고 적용 여부를 결정합니다.

6. 마켓 전송

편집이 완료된 상품을 각 마켓으로 전송합니다.

상품 전송

  1. 전송할 상품을 선택합니다.
  2. 상품전송 버튼을 클릭합니다.
  3. 전송할 마켓과 계정을 선택합니다.
  4. 카테고리 매핑이 완료되었는지 확인합니다.
  5. 전송 시작을 클릭합니다.

마켓 삭제

마켓에 등록된 상품을 삭제합니다. 마켓 API를 통해 실제 마켓에서 상품이 삭제됩니다.

강제 삭제

마켓 API 연동 없이 삼바웨이브 내에서만 전송 기록을 삭제합니다. 마켓에서 이미 수동으로 삭제한 상품의 기록을 정리할 때 사용합니다.

TIP 전송 전 반드시 카테고리 매핑과 정책 적용이 완료되었는지 확인하세요. 미완료 시 전송 오류가 발생합니다.
