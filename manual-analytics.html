1. 통계 대시보드

매출통계 페이지 상단에 핵심 지표가 요약 카드로 표시됩니다.

주요 지표

--
총 매출
--
총 수익
--
주문 건수
--%
평균 수익률

대시보드 구성

  • 매출 추이 그래프 — 일별/주별/월별 매출 변화를 시각적으로 확인
  • 요약 카드 — 총 매출, 순이익, 주문 건수, 수익률 핵심 수치
  • 마켓별 비중 — 도넛 차트로 마켓별 매출 비중 확인
  • 인기 상품 TOP 10 — 가장 많이 판매된 상품 순위
[스크린샷] 통계 대시보드 — 상단 요약 카드 4개, 매출 추이 라인 차트, 마켓별 비중 도넛 차트
TIP 대시보드는 선택된 기간에 따라 데이터가 변경됩니다. 상단의 기간 필터를 활용하여 원하는 기간의 데이터를 확인하세요.

2. 소싱처별 / 마켓별 분석

소싱처와 판매 마켓을 기준으로 매출을 분석합니다.

소싱처별 분석

어떤 소싱처에서 수집한 상품이 가장 수익성이 높은지 확인합니다.

  • 소싱처 — 무신사, KREAM, SSG 등
  • 총 매출 — 해당 소싱처 상품의 총 판매 금액
  • 원가 합계 — 소싱처 매입 원가 합계
  • 순이익 — 매출에서 원가와 수수료를 제외한 순수 이익
  • 수익률 — 순이익 / 매출 x 100
  • 판매 건수 — 해당 소싱처 상품의 총 판매 건수
[스크린샷] 소싱처별 매출 테이블 — 소싱처명, 매출, 원가, 순이익, 수익률, 판매건수 컬럼

마켓별 분석

어떤 마켓에서 가장 많은 매출이 발생하는지 확인합니다.

  • 마켓 — 스마트스토어, 쿠팡, 11번가 등
  • 총 매출 — 해당 마켓의 총 판매 금액
  • 수수료 — 마켓에 지불한 수수료 합계
  • 순이익 — 마켓별 순수 이익
  • 판매 건수 — 해당 마켓의 총 주문 건수
  • 평균 주문 금액 — 주문 1건당 평균 결제 금액
[스크린샷] 마켓별 매출 테이블 및 비교 막대 차트
TIP 수익률이 낮은 마켓이나 소싱처는 정책(마진율, 배송비)을 재조정하여 수익성을 개선할 수 있습니다.

3. 기간별 필터

조회 기간을 설정하여 원하는 기간의 매출 데이터를 분석합니다.

기간 선택 옵션

  • 오늘 — 당일 매출 확인
  • 1주 — 최근 7일간 매출
  • 1개월 — 최근 30일간 매출
  • 3개월 — 최근 90일간 매출 (분기 분석)
  • 6개월 — 최근 180일간 매출 (반기 분석)
  • 1년 — 최근 365일간 매출 (연간 분석)
  • 직접입력 — 시작일과 종료일을 직접 설정
[스크린샷] 기간 필터 버튼 — 오늘 / 1주 / 1개월 / 3개월 / 6개월 / 1년 / 직접입력 버튼 그룹

기간별 비교

이전 기간과의 비교를 통해 매출 추세를 파악할 수 있습니다.

  • 전주 대비 매출 증감률
  • 전월 대비 매출 증감률
  • 전년 동기 대비 매출 증감률
참고 통계 데이터는 주문 완료(배송완료) 기준으로 집계됩니다. 취소/반품 건은 제외됩니다.

4. 수익률 계산

삼바웨이브에서 사용하는 수익률 계산 방식입니다.

기본 수익률 공식

수익률 (%) = (순이익 / 매출) x 100

순이익 계산

순이익 = 판매가 - 원가 - 마켓수수료 - 배송비

계산 예시

판매가 80,000원, 원가 50,000원, 마켓수수료 4,800원(6%), 배송비 3,000원인 경우:

순이익 = 80,000 - 50,000 - 4,800 - 3,000 = 22,200원
수익률 = (22,200 / 80,000) x 100 = 27.75%

수익률 목표

  • 20% 이상 — 양호한 수익률
  • 15~20% — 보통 수준, 개선 여지 있음
  • 10~15% — 낮은 수익률, 정책 재검토 필요
  • 10% 미만 — 적자 위험, 긴급 정책 조정 필요
TIP 수익률이 낮다면 마진율을 올리거나 수수료가 낮은 마켓을 우선적으로 활용하세요. 소싱처 원가가 낮은 상품을 발굴하는 것도 효과적입니다.
