반품 및 교환 요청을 관리하고 처리하는 방법

SambaWave Docs > 반품교환 매뉴얼

1. 반품교환 목록

마켓에서 접수된 반품/교환 요청을 목록으로 확인합니다.

테이블 컬럼 설명

  • 요청일 — 고객이 반품/교환을 요청한 날짜
  • 주문번호 — 원래 주문의 고유번호
  • 마켓 — 주문이 발생한 판매 마켓
  • 상품명 — 반품/교환 대상 상품명
  • 옵션 — 선택된 옵션 정보
  • 유형 — 반품 또는 교환 구분
  • 사유 — 고객이 선택한 반품/교환 사유
  • 상태 — 현재 처리 상태 (요청/승인/수거중/완료/거절)
  • 처리일 — 처리가 완료된 날짜
[스크린샷] 반품교환 목록 테이블 — 요청일, 주문번호, 마켓, 상품명, 유형, 사유, 상태 컬럼이 표시된 목록

2. 기간/마켓 필터

반품교환 목록을 조건별로 필터링합니다.

필터 옵션

  • 기간 필터 — 요청일 기준으로 조회 기간을 설정합니다.
    • 오늘 / 1주 / 1개월 / 3개월 / 직접입력
  • 마켓 필터 — 특정 마켓의 반품/교환만 표시합니다.
  • 유형 필터 — 반품만 보기, 교환만 보기를 선택합니다.
  • 상태 필터 — 요청/승인/수거중/완료/거절 상태별로 필터링합니다.
[스크린샷] 필터 영역 — 기간 선택 버튼, 마켓 드롭다운, 유형/상태 필터 버튼
TIP 반품/교환 처리 기한을 놓치지 않으려면 "요청" 상태를 기본 필터로 설정하여 미처리 건을 우선 확인하세요.

3. 반품 상태 관리

반품/교환 요청을 단계별로 처리합니다.

처리 흐름

요청
→
승인
→
수거중
→
완료 (환불/교환발송)
거절

반품 승인

  1. "요청" 상태의 반품 건을 확인합니다.
  2. 반품 사유와 상품 상태를 검토합니다.
  3. 승인 버튼을 클릭합니다.
  4. 수거 방법(택배 회수/직접 반송)을 선택합니다.
  5. 승인이 완료되면 상태가 "수거중"으로 변경됩니다.

반품 완료

  1. 반품 상품이 도착했는지 확인합니다.
  2. 상품 상태를 검수합니다.
  3. 완료 버튼을 클릭합니다.
  4. 환불 처리가 자동으로 진행됩니다.

반품 거절

  1. 거절 사유가 명확한 경우 거절 버튼을 클릭합니다.
  2. 거절 사유를 입력합니다. (예: "사용 흔적이 있어 반품 불가")
  3. 고객에게 거절 사유가 자동으로 통보됩니다.
주의 반품 거절 시 마켓 정책에 따라 패널티가 발생할 수 있습니다. 거절 전 마켓별 반품 규정을 반드시 확인하세요.

4. 로그 확인

반품/교환 처리 이력을 확인합니다.

로그 확인 방법

  • 반품/교환 건을 클릭하면 상세 이력 모달이 표시됩니다.
  • 각 단계별 처리일시, 처리자, 메모를 확인할 수 있습니다.
  • 고객과의 커뮤니케이션 이력도 함께 표시됩니다.
[스크린샷] 반품교환 상세 모달 — 처리 이력 타임라인, 메모, 상태 변경 기록

통계 확인

반품/교환 현황을 통계로 확인할 수 있습니다.

  • 기간별 반품률
  • 마켓별 반품 건수
  • 사유별 반품 분포
참고 반품률이 높은 상품은 소싱 중단을 고려하세요. 반품률이 일정 수준을 넘으면 마켓에서 판매 제한을 받을 수 있습니다.
