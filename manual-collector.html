1. 소싱처 선택

상품수집 페이지 상단에 소싱처 버튼이 나열되어 있습니다. 원하는 소싱처를 클릭하여 해당 사이트의 상품을 수집할 수 있습니다.

지원 소싱처 목록

  • 무신사 (MUSINSA) — 국내 패션 플랫폼, 의류/신발/액세서리
  • KREAM — 한정판 스니커즈/리셀 플랫폼
  • SSG (신세계몰) — 종합 쇼핑몰
  • 롯데ON — 종합 쇼핑몰
  • 다나와 — 가격비교 사이트
  • 스마트스토어 — 네이버 스마트스토어 소싱
  • G마켓 — 오픈마켓 소싱
소싱처 선택
무신사 KREAM 다나와 신세계몰 롯데ON 스마트스토어 G마켓
  1. 상품수집 메뉴에 진입합니다.
  2. 상단 소싱처 버튼 중 원하는 사이트를 클릭합니다.
  3. 선택된 소싱처에 맞는 URL 입력 필드와 옵션이 표시됩니다.
TIP KREAM은 확장앱 기반 큐 방식으로 수집됩니다. 확장앱이 설치되어 있어야 합니다.

2. URL로 수집하기

소싱처의 검색 결과 페이지 URL 또는 단일 상품 URL을 입력하여 상품을 수집합니다.

검색 URL 수집

  1. 소싱처 사이트에서 원하는 카테고리 또는 검색 결과 페이지로 이동합니다.
  2. 브라우저 주소창의 URL을 복사합니다.
  3. 삼바웨이브 상품수집 페이지의 URL 입력란에 붙여넣습니다.
  4. 수집 시작 버튼을 클릭합니다.
  5. 해당 페이지의 모든 상품이 자동으로 수집됩니다.
URL 입력
https://www.musinsa.com/categories/001 ...
수집 시작

단일 상품 URL 수집

특정 상품 하나만 수집하고 싶을 때 해당 상품의 상세 페이지 URL을 입력합니다.

참고 무신사 수집 시 동시 요청 수(concurrency)가 1로 제한됩니다. 대량 수집 시 시간이 소요될 수 있습니다.

3. 수집 옵션

수집 시 적용할 수 있는 필터 옵션을 설정합니다.

옵션 체크박스

  • 예약배송 제외 — 예약배송 상품을 수집 대상에서 제외합니다. 즉시 발송 가능한 상품만 수집할 때 체크합니다.
  • 부티끄 제외 — 부티끄(고가 럭셔리) 상품을 제외합니다. 일반 상품만 수집할 때 사용합니다.
  • 최대혜택가 적용 — 쿠폰/포인트 등 최대 혜택이 적용된 가격으로 원가를 설정합니다.
수집 옵션
TIP 예약배송 상품은 배송이 지연될 수 있어 위탁판매에 부적합합니다. 가급적 제외하는 것을 권장합니다.

4. 수집 그룹 관리

수집된 상품을 체계적으로 관리하기 위해 그룹(폴더) 구조를 사용합니다.

그룹 생성

  1. 좌측 트리 패널에서 + 새 그룹 버튼을 클릭합니다.
  2. 그룹 이름을 입력합니다. (예: "무신사 스니커즈", "KREAM 한정판")
  3. 하위 그룹이 필요하면 상위 그룹을 우클릭하여 하위 폴더를 생성합니다.

그룹에 정책 적용

그룹에 정책을 연결하면 해당 그룹에 속한 모든 상품에 가격정책이 자동 적용됩니다.

  1. 그룹을 선택합니다.
  2. 정책 드롭다운에서 적용할 정책을 선택합니다.
  3. 정책이 적용되면 해당 그룹의 상품 판매가가 자동 계산됩니다.

5. 수집 상품 목록

수집된 상품은 무한스크롤 방식으로 표시됩니다. 10만 건 이상의 대량 데이터도 쾌적하게 로드됩니다.

필터 및 검색

  • 상품명 검색 — 상품명 키워드로 검색합니다.
  • 수집일 필터 — 특정 기간에 수집된 상품만 표시합니다.
  • 소싱처 필터 — 특정 소싱처의 상품만 표시합니다.
  • 상태 필터 — 수집 완료/진행 중/실패 상품을 구분합니다.

상품 선택 및 일괄 작업

  1. 체크박스로 개별 상품을 선택하거나, 전체 선택을 클릭합니다.
  2. 선택된 상품에 대해 그룹 이동, 삭제, 정책 적용 등의 일괄 작업을 실행합니다.

6. AI 소싱기

키워드를 입력하면 AI가 자동으로 관련 상품을 수집하는 기능입니다.

  1. AI 소싱 탭을 클릭합니다.
  2. 수집할 키워드를 입력합니다. (예: "나이키 에어맥스", "아디다스 스탠스미스")
  3. 소싱처를 선택합니다. (복수 선택 가능)
  4. 소싱 시작 버튼을 클릭하면 AI가 관련 상품을 자동 검색하고 수집합니다.
