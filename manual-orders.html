1. PlayAuto 연동 주문 동기화

PlayAuto를 통해 여러 마켓의 주문을 삼바웨이브 한 곳에서 자동으로 동기화합니다.

PlayAuto 연동 개요

  • PlayAuto에 연결된 모든 마켓 주문이 자동으로 수집됩니다.
  • 주문 상태 변경(발주확인, 송장등록 등)이 PlayAuto를 통해 마켓에 반영됩니다.
  • 반품/교환 정보도 PlayAuto를 통해 동기화됩니다.

PlayAuto 지원 마켓

  • 스마트스토어, 쿠팡, 11번가, 롯데ON, SSG, GS샵, G마켓/옥션(ESMPlus), 카페24 등 PlayAuto 연동 마켓 전체
참고 PlayAuto 계정 연동은 설정 페이지에서 완료해야 합니다. PlayAuto 연동 후 주문 동기화 주기는 설정에서 조정할 수 있습니다.

2. 주문 가져오기

각 마켓에서 새로운 주문을 가져옵니다.

지원 마켓

  • 스마트스토어, 쿠팡, 11번가, 롯데ON, SSG, GS샵, G마켓/옥션(ESMPlus), 카페24
  • PlayAuto 연동 마켓 전체 (PlayAuto를 통해 일괄 수집 가능)

개별 마켓 가져오기

  1. 주문관리 페이지 상단의 가져오기 버튼을 클릭합니다.
  2. 주문을 가져올 마켓을 선택합니다.
  3. 기간을 설정합니다. (기본: 최근 7일)
  4. 가져오기 실행을 클릭합니다.

전체 마켓 가져오기

전체마켓 가져오기 버튼을 클릭하면 연결된 모든 마켓에서 한 번에 주문을 수집합니다.

전체마켓 가져오기 스마트스토어 쿠팡 11번가
오늘 1주일 1개월
TIP 하루에 한 번 이상 주문을 가져오는 것을 권장합니다. 주문 처리가 지연되면 마켓 패널티가 발생할 수 있습니다.

3. 주문 목록

가져온 주문을 목록으로 확인합니다.

필터 옵션

  • 기간 필터 — 주문일 기준으로 기간을 설정합니다. (오늘, 1주, 1개월, 직접입력)
  • 마켓별 필터 — 특정 마켓의 주문만 표시합니다.
  • 상태 필터 — 신규주문, 발주확인, 배송중, 배송완료 등 상태별 필터링
  • 검색 — 주문번호, 상품명, 구매자명으로 검색합니다.

5. 주문 처리

주문을 단계별로 처리합니다.

주문 처리 흐름

신규주문
→
발주확인
→
발주확정
→
배송준비중
→
배송중
→
배송완료

주문 상태 설명

  • 신규주문 — 마켓에서 수집된 미처리 주문
  • 발주확인 — 소싱처에 주문 접수 완료
  • 발주확정 — 소싱처에서 주문 확정 처리
  • 배송준비중 — 소싱처 출고 준비 단계
  • 배송중 — 송장 등록 및 배송 진행 중
  • 배송완료 — 고객 수령 완료
  • 취소요청 — 고객 또는 마켓의 취소 요청
  • 취소완료 — 취소 처리 완료
  • 반품요청 — 고객 반품 요청 접수

발주확인

  1. 신규주문 상태의 주문을 선택합니다.
  2. 소싱처에서 해당 상품을 주문합니다. (무신사, KREAM 등)
  3. 발주확인 버튼을 클릭하여 발주 완료를 기록합니다.

송장등록

  1. 소싱처에서 발송된 상품의 송장번호를 확인합니다.
  2. 주문의 송장등록 버튼을 클릭합니다.
  3. 택배사를 선택하고 송장번호를 입력합니다.
  4. 등록을 클릭하면 마켓에 자동으로 송장이 전송됩니다.

배송추적

등록된 송장번호로 배송 상태를 실시간 추적할 수 있습니다.

TIP 발주확인과 송장등록은 마켓별 처리 기한이 있습니다. 스마트스토어의 경우 발주확인 2일, 송장등록 3일 이내 처리가 필요합니다.

6. 인증 오류 해결

마켓 쿠키 또는 API 인증이 만료된 경우 주문 수집이 실패할 수 있습니다.

쿠키 만료 시 해결 방법

  1. 크롬 확장앱을 통해 해당 마켓에 로그인합니다.
  2. 확장앱이 자동으로 쿠키를 갱신합니다.
  3. 주문 가져오기를 다시 실행합니다.

API 키 만료 시 해결 방법

  1. 설정 페이지에서 해당 마켓의 API 연결 설정으로 이동합니다.
  2. 마켓 판매자센터에서 새 API 키를 발급받습니다.
  3. 설정 페이지에서 API 키를 업데이트합니다.
TIP 마켓 로그인 세션은 일정 기간 후 만료됩니다. 주기적으로 확장앱을 통해 마켓에 접속하여 세션을 유지하세요.

7. 외부 링크 활용

주문과 관련된 외부 사이트로 바로 이동할 수 있습니다.

링크 종류

  • 원문링크 — 소싱처의 원본 상품 페이지로 이동합니다. 재고 확인, 가격 변동 확인에 활용합니다.
  • 판매마켓링크 — 마켓에 등록된 상품 페이지로 이동합니다. 등록 상태를 확인합니다.
  • 다나와 검색 — 다나와에서 해당 상품을 검색합니다. 최저가 비교에 활용합니다.
  • 네이버 검색 — 네이버에서 해당 상품을 검색합니다.
외부 링크
원문링크 판매마켓 다나와 검색 네이버 검색

8. SMS / 카카오 메시지 발송

고객에게 주문 관련 알림 메시지를 발송합니다.

SMS 발송

  1. 메시지를 발송할 주문을 선택합니다.
  2. SMS 발송 버튼을 클릭합니다.
  3. 메시지 템플릿을 선택하거나 직접 작성합니다.
  4. 발송을 클릭합니다.

카카오 알림톡 발송

카카오 비즈니스 채널을 통해 알림톡을 발송합니다. 발송 성공률이 SMS보다 높고 비용이 저렴합니다.

참고 SMS/카카오 메시지 발송에는 별도 요금이 부과됩니다. 설정에서 발송 서비스 연결이 필요합니다.
