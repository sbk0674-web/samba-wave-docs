카테고리매핑 매뉴얼

소싱처 카테고리를 마켓 카테고리에 연결하는 방법

1. 카테고리 매핑이란

소싱처(무신사, KREAM 등)의 카테고리 체계와 판매 마켓(스마트스토어, 쿠팡 등)의 카테고리 체계는 서로 다릅니다.

카테고리 매핑은 소싱처의 카테고리를 마켓의 카테고리에 연결하는 작업입니다. 매핑이 완료되어야 상품을 마켓에 전송할 수 있습니다.

소싱처 (무신사)
상의 > 반소매 티셔츠
→
마켓 (스마트스토어)
패션의류 > 남성의류 > 티셔츠
소싱처 (무신사)
신발 > 스니커즈
→
마켓 (쿠팡)
패션 > 남성신발 > 스니커즈
참고 카테고리 매핑이 완료되지 않은 상품은 마켓에 전송할 수 없습니다. 전송 전 반드시 매핑을 완료하세요.

2. 카테고리 매핑하기

소싱처에서 수집된 카테고리를 각 마켓의 카테고리에 연결합니다.

  1. 좌측 메뉴에서 카테고리매핑을 클릭합니다.
  2. 매핑할 소싱처를 선택합니다. (예: 무신사)
  3. 소싱처 카테고리 목록에서 매핑할 카테고리를 선택합니다.
  4. 우측에 전송할 마켓을 선택합니다. (예: 스마트스토어)
  5. 마켓의 카테고리 트리에서 매핑할 카테고리를 찾아 선택합니다.
  6. 매핑 저장 버튼을 클릭합니다.

3. 일괄 매핑

여러 소싱처 카테고리를 한 번에 동일한 마켓 카테고리에 매핑할 수 있습니다.

  1. 소싱처 카테고리 목록에서 체크박스로 여러 카테고리를 선택합니다.
  2. 우측 마켓 카테고리 트리에서 매핑할 카테고리를 선택합니다.
  3. 일괄 매핑 버튼을 클릭합니다.
  4. 확인 모달에서 매핑 내용을 검토하고 적용을 클릭합니다.
일괄 매핑 확인
선택된 소싱처 카테고리 (3건)
상의 > 반소매 티셔츠
상의 > 긴소매 티셔츠
상의 > 민소매 티셔츠
매핑 대상 마켓 카테고리
패션의류 > 남성의류 > 티셔츠
취소 적용
TIP 유사한 카테고리(예: "반소매 티셔츠", "긴소매 티셔츠", "민소매 티셔츠")는 일괄 매핑으로 한 번에 처리하면 효율적입니다.

4. 매핑 확인 및 수정

기존 매핑을 확인하고 수정하는 방법입니다.

매핑 현황 확인

  • 카테고리매핑 페이지에서 이미 매핑된 카테고리는 연결된 마켓 카테고리가 표시됩니다.
  • 미매핑 카테고리는 별도로 표시되어 쉽게 확인할 수 있습니다.
  • 마켓별로 필터링하여 특정 마켓의 매핑 현황만 볼 수 있습니다.

매핑 수정

  1. 수정할 매핑 항목을 클릭합니다.
  2. 새로운 마켓 카테고리를 선택합니다.
  3. 수정 저장 버튼을 클릭합니다.

매핑 삭제

잘못된 매핑은 해당 항목의 삭제 버튼을 클릭하여 제거할 수 있습니다.

소싱처 카테고리 마켓 카테고리 상태 작업
상의 > 반소매 티셔츠 패션의류 > 남성의류 > 티셔츠 매핑완료 수정
신발 > 스니커즈 패션 > 남성신발 > 스니커즈 매핑완료 수정
가방 > 백팩 미설정 미매핑 매핑
참고 매핑을 수정하면 해당 카테고리에 속한 모든 상품의 마켓 카테고리가 자동으로 업데이트됩니다.
