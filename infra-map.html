5. API 키 어디서 관리하는지

서비스키 이름저장 위치설정 방법
GeminiAPI KeyDB (samba_settings.gemini)설정 페이지 → AI 설정
ClaudeAPI KeyDB (samba_settings.claude)설정 페이지 → AI 설정
R2Access/Secret KeyDB (samba_settings.cloudflare_r2)설정 페이지 → 스토리지
AligoAPI Key + User IDDB (samba_settings.aligo_sms)설정 페이지 → 알림
스마트스토어Client ID/SecretDB (마켓 계정)설정 페이지 → 스토어 연결
쿠팡Access/Secret KeyDB (마켓 계정)설정 페이지 → 스토어 연결
무신사쿠키 (자동)DB (samba_settings.musinsa_cookie)확장앱이 자동 전송
핵심: 모든 API 키는 설정 페이지에서 관리됩니다. 서버 환경변수나 코드에 직접 넣지 않습니다.
Railway/Vercel 대시보드에는 DB 연결 정보와 JWT 시크릿만 설정하면 됩니다.

6. 배포는 어떻게 되는지

코드 수정 → git push main → GitHub Actions (CI) → Vercel 자동배포 + Railway 자동배포

push 한 번이면 프론트+백엔드 동시에 자동 배포됩니다. 별도 작업 불필요.
