고객 문의를 수집하고 답변을 관리하는 방법

1. CS 문의 가져오기

각 마켓에서 접수된 고객 문의를 삼바웨이브로 가져옵니다.

전체마켓 가져오기

  1. CS관리 페이지 상단의 전체마켓 가져오기 버튼을 클릭합니다.
  2. 연결된 모든 마켓에서 미답변 문의를 자동으로 수집합니다.
  3. 수집이 완료되면 새로운 문의가 목록에 추가됩니다.

스마트스토어 상품문의 동기화

스마트스토어의 상품문의는 API를 통해 실시간으로 동기화됩니다.

  • 상품에 대한 일반 문의
  • 배송 관련 문의
  • 교환/반품 관련 문의
  • 기타 문의
[스크린샷] CS 문의 가져오기 버튼 — "전체마켓 가져오기" 버튼과 마켓별 동기화 상태 표시
TIP 문의 응답 시간이 빠를수록 마켓 평점에 긍정적입니다. 하루 2~3회 문의를 확인하는 것을 권장합니다.

2. 문의 목록

수집된 문의를 목록으로 확인하고 관리합니다.

테이블 컬럼 설명

  • 마켓 — 문의가 접수된 판매 마켓 (스마트스토어, 쿠팡 등)
  • 주문번호 — 관련 주문의 번호 (주문 관련 문의인 경우)
  • 문의유형 — 상품문의, 배송문의, 교환/반품, 기타
  • 고객 — 문의한 고객의 아이디 또는 이름
  • 문의내용 — 고객이 작성한 문의 내용 (미리보기)
  • 답변여부 — 미답변 / 답변완료 상태 표시
  • 문의일시 — 문의가 등록된 날짜와 시간
[스크린샷] CS 문의 목록 테이블 — 마켓, 주문번호, 문의유형, 고객, 문의내용, 답변여부 컬럼이 표시된 목록

필터 및 검색

  • 답변여부 필터 — 미답변만 보기 / 답변완료만 보기
  • 마켓 필터 — 특정 마켓의 문의만 표시
  • 문의유형 필터 — 상품문의, 배송문의 등 유형별 필터
  • 키워드 검색 — 문의 내용, 고객명, 주문번호로 검색

3. 답변 등록

고객 문의에 답변을 작성하고 등록합니다.

  1. 미답변 문의를 클릭하여 상세 내용을 확인합니다.
  2. 답변 버튼을 클릭하면 답변 모달이 표시됩니다.
  3. 답변 템플릿을 선택하거나 직접 답변을 작성합니다.
  4. 작성된 답변 내용을 검토합니다.
  5. 답변 전송 버튼을 클릭합니다.
[스크린샷] 답변 모달 — 문의 내용 표시, 템플릿 선택 드롭다운, 답변 작성 텍스트 영역, 전송 버튼

답변 작성 요령

  • 고객의 문의 내용을 정확히 파악한 후 답변합니다.
  • 예상 배송일, 재입고 일정 등 구체적인 정보를 포함합니다.
  • 정중하고 친절한 어투를 사용합니다.
  • 연락처나 개인정보는 답변에 포함하지 않습니다.

4. 답변 전송

작성된 답변을 마켓에 직접 전송합니다.

스마트스토어 답변 전송

스마트스토어의 상품문의에 대한 답변은 커머스 API를 통해 스마트스토어에 직접 등록됩니다.

  1. 답변 모달에서 답변 전송을 클릭합니다.
  2. 스마트스토어 API로 답변이 전송됩니다.
  3. 전송 성공 시 답변여부가 "답변완료"로 변경됩니다.
  4. 스마트스토어 판매자 센터에서 답변이 정상 등록되었는지 확인할 수 있습니다.
참고 현재 답변 직접 전송은 스마트스토어만 지원됩니다. 다른 마켓은 답변 내용을 복사하여 해당 마켓 판매자 센터에서 직접 등록합니다.
TIP 답변 전송에 실패한 경우, 네트워크 상태와 API 인증 정보를 확인하세요. 설정 페이지에서 스마트스토어 연결 상태를 재확인할 수 있습니다.

5. 답변 템플릿 관리

자주 사용하는 답변을 템플릿으로 저장하여 빠르게 답변합니다.

템플릿 생성

  1. CS관리 페이지에서 템플릿 관리 버튼을 클릭합니다.
  2. + 새 템플릿을 클릭합니다.
  3. 템플릿 이름과 답변 내용을 작성합니다.
  4. 저장을 클릭합니다.

템플릿 예시

배송 안내 템플릿
안녕하세요, [스토어명]입니다.
문의해 주셔서 감사합니다.

해당 상품은 주문 확인 후 영업일 기준 2~3일 내 출고 예정이며,
출고 후 1~2일 내 배송 완료됩니다.

배송 관련 추가 문의사항이 있으시면 언제든 문의해 주세요.
감사합니다.
품절 안내 템플릿
안녕하세요, [스토어명]입니다.
문의해 주셔서 감사합니다.

죄송합니다. 해당 상품/옵션은 현재 품절 상태입니다.
재입고 시 상품 페이지에 안내드리겠습니다.

불편을 드려 죄송합니다. 감사합니다.

템플릿 활용

답변 모달에서 템플릿 드롭다운을 선택하면 저장된 답변이 자동으로 채워집니다. 필요에 따라 내용을 수정한 후 전송할 수 있습니다.

TIP 문의 유형별로 템플릿을 준비해 두면 답변 속도가 크게 향상됩니다. 배송, 품절, 교환/반품, 사이즈 문의 등 카테고리별로 2~3개씩 만들어 두세요.
