1. 마켓 스토어 연결

상품 전송과 주문 수집을 위해 각 마켓의 판매자 계정을 연결합니다.

  1. 좌측 메뉴에서 설정을 클릭합니다.
  2. 연결할 마켓의 카드를 찾습니다.
  3. 연결하기 버튼을 클릭합니다.
  4. API 인증 정보를 입력합니다. (마켓마다 다름)
  5. 저장을 클릭합니다.

2. API 키 발급 방법

각 마켓의 API 키를 발급받는 방법입니다.

스마트스토어

1. 네이버 커머스 API 센터 (https://apicenter.commerce.naver.com) 접속

2. 애플리케이션 등록 > 새 애플리케이션 생성

3. Client ID, Client Secret 발급

4. 삼바웨이브 설정에 입력

쿠팡

1. 쿠팡 윙 (wing.coupang.com) > 판매자 정보 > API 관리

2. Secret Key, Access Key 발급

3. Vendor ID 확인

4. 삼바웨이브 설정에 입력

11번가

1. 11번가 셀러오피스 > Open API 관리

2. API Key 발급 신청

3. 승인 후 API Key 확인

4. 삼바웨이브 설정에 입력

롯데ON

1. 롯데ON 셀러센터 > API 관리

2. API 인증 토큰 발급

3. 삼바웨이브 설정에 입력

SSG (신세계몰)

1. SSG 파트너센터 > API 관리

2. 인증 토큰 발급

3. 삼바웨이브 설정에 입력

* SSG는 계약 브랜드만 등록 가능합니다.

아마존 / 라쿠텐 / 바이마

해외 마켓은 각 마켓의 셀러 센터에서 API 키를 발급받습니다.

발급 절차가 마켓마다 다르므로 각 마켓의 공식 가이드를 참고하세요.

주의 API 키는 절대로 타인과 공유하지 마세요. 유출 시 판매자 계정이 위험해질 수 있습니다.

3. 인증 테스트

API 키를 입력한 후 연결이 정상적인지 테스트합니다.

  1. API 인증 정보를 입력하고 저장합니다.
  2. 인증 테스트 버튼을 클릭합니다.
  3. 연결 성공 시 녹색 "연결됨" 표시가 나타납니다.
  4. 연결 실패 시 에러 메시지를 확인하고 정보를 재확인합니다.

자주 발생하는 인증 오류

  • Invalid API Key — API 키가 잘못 입력됨. 복사-붙여넣기로 재입력
  • Expired Token — 인증 토큰이 만료됨. 마켓에서 재발급 필요
  • Permission Denied — API 권한 부족. 마켓에서 필요 권한 추가 설정
  • Network Error — 네트워크 연결 문제. 인터넷 연결 확인

4. 공통 설정

모든 마켓에 공통으로 적용되는 기본 설정을 관리합니다.

A/S 안내

마켓에 등록되는 상품의 A/S 안내 정보를 설정합니다.

  • A/S 전화번호
  • A/S 안내 문구
  • 교환/반품 주소

배송비 설정

  • 무료배송 기준 금액 — 이 금액 이상 구매 시 무료배송
  • 기본 배송비 — 무료배송 기준 미달 시 부과되는 배송비
  • 도서/산간 추가 배송비 — 도서산간 지역 추가 배송비

재고 설정

  • 기본 재고 수량 — 마켓에 표시할 기본 재고 수량
  • 품절 시 자동 판매중지 — 소싱처 품절 시 자동으로 판매 중지 여부

할인/혜택 설정

  • 포인트 적립률 — 고객에게 제공할 포인트 적립 비율
  • 리뷰 혜택 — 리뷰 작성 시 제공할 혜택 (포인트, 쿠폰 등)

5. 설정 저장

변경된 설정을 저장합니다.

  1. 설정 항목을 수정합니다.
  2. 페이지 하단의 설정 저장 버튼을 클릭합니다.
  3. 저장 성공 알림이 표시됩니다.
참고 설정 변경 후 반드시 저장 버튼을 클릭해야 합니다. 페이지를 이동하면 저장되지 않은 변경사항은 사라집니다.
